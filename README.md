# ミンラボ 会社ホームページ

合同会社ミンラボの公式ホームページ（旧Wixサイトの作り直し）。
ビルド不要の静的HTML（`partials/` の共通ヘッダー・フッターを `js/include.js` が読み込む方式）。
議員としての林みきアカウント（GitHub/Netlifyなど）とは完全に分離して運用する。

## 構成

- `index.html` — ホーム（理念文）／事業（5カテゴリ）／メンバー／お問い合わせ
- `about.html` — 会社概要

## デプロイ手順（Cloudflare Pages）

このリポジトリ自体はどこにも接続されていません。以下はユーザー側で行う作業です。

1. GitHubでミンラボ用の新規アカウント（またはOrganization）を作成（`minnlabo-portal`と共通のアカウントでよい）
2. このディレクトリの内容を新規リポジトリとしてpush
3. Cloudflareアカウントを新規作成（またはログイン）
4. Cloudflare Pages → 「Gitに接続」で上記リポジトリを選択
5. ビルド設定：
   - Framework preset: `None`
   - Build command: 空欄のまま
   - Build output directory: `/`
6. デプロイすると `https://<プロジェクト名>.pages.dev` で公開される

独自ドメインを取得した場合は、Cloudflare Pagesのカスタムドメイン設定から追加できます。
